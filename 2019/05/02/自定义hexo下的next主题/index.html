重装系统后重新配置hexo连接github pages（需预先保存blog文件夹）

打开.ssh文件夹，id_rsa.pub文件，
在GitHub账号的setting中，找到SSH keys的设置选项，
点击New SSH key，把id_rsa.pub里面的信息复制进去。
